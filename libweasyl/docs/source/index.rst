===========
 libweasyl
===========

To cope with python's three string types,
this document uses the following naming conventions:

.. glossary::

   native string
     ``str`` in python 2 and ``str`` in python 3.

   unicode
     A string of unicode code points.
     ``unicode`` in python 2 and ``str`` in python 3.

   bytes
     A string of bytes.
     ``str`` in python 2 and ``bytes`` in python 3.
